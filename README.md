# ✨ Lucky Letter Game – Python Mini Project 🐍

## 🚀 About This Project

This is a fun beginner-friendly Python mini game where the user enters their name and the program checks lucky letters to give a fun prediction.

---

## 🎮 How the Game Works

1. User enters their name
2. Program converts the name into lowercase
3. Counts letters like `a` and `e`
4. Compares the counts
5. Displays a fun result
